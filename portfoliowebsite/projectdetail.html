User Interface

Project 1

This project was inspired by Umamusume, a popular mobile game and anime series. My goal was to create a user interface that can be used in a Roblox game, serving as a shop for players to buy different items and characters that captures the essence of the game while providing a seamless experience for the user.

Project 2

This project was inspired by a Roblox game called TYPE://SOUL, a popular game inspired by the anime Bleach. I was trying to replicate the stats menu in the game, used to upgrade a character's skill points. It was a fun project to work on, and I learned about how to properly use background textures to improve the visual appeal of the interface.

Project 3

This project was made for a client, in which they wanted a UI is that was inspired by a Roblox game named Overkill. The UI is supposed to be an timed item shop, in which players can purchase various items. I like the shade of blue in the background, as I think it looks pretty great.

Coding Projects

Project 1

This project is a sample of my larger advanced throwable ball system, utilzing physics as a way to create realistic movement while the ball is in movement. It utilizes gravity and a modular speed variable to ensure a nice, unique ball system. It was challenging for me to implement, but I am proud of the result and it helped me understand physics in the context of game development.

Project 2

This project is a random coordinate point system where a random number for the X and Y axis is generated anywhere from 0 to 100 inside of a rectangle. The rectangle is supposed to be cenetered at the origin. It was relatively easy to implement, but there were some challenges with ensuring the points were distributed evenly.

Project 3

This project is a triangle generation system where the compiler would calculate information of a triangle based on certain parameters. It was a good exercise in understanding geometric calculations and how to implement them in code, forcing me to use Algebra skills that I did not think I would have to useoutside of school.
